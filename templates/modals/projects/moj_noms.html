MoJ NOMS - Strategic Partner Gateway

Lead developer within a team responsible for building a message routing system to integrate the MoJ NOMS's central systems with external partner systems. Taleb was part of the initial team that kick-started the agile delivery lifecycle, creating all the main building blocks, making crucial implementation decisions and driving the delivery of a secure, scalable and responsive system. The system was built using Apache Camel, and integrated with SOAP, JAX-RS and ActiveMQ endpoints.
